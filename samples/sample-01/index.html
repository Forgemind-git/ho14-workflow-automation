Hands-On 14 · Sample 01

Daily Web Scraper → Sheet

Automatically scrape a website every day and append fresh data to a spreadsheet — no manual copying, no missed updates.

Subscription required

Claude.ai Pro or Team plan with Cowork access. No API key or local Python setup needed.

What this does

This automation visits a target website on a daily schedule, extracts specific data elements (prices, headlines, quotes — whatever matches your CSS selector), and appends each value as a timestamped row to a CSV spreadsheet.

  • Visits the page with a proper User-Agent header
  • Extracts all elements matching the configured CSS selector
  • Cleans values (strips whitespace and quote characters)
  • Appends timestamped rows to a CSV file
  • Runs every morning so data is ready before you start work
Subscription required

This guide uses Claude Cowork — a scheduled automation feature available on the Claude.ai Pro and Team plans.

  • Go to claude.ai and sign in
  • You need a Pro or Team subscription
  • Cowork lets you schedule recurring prompts — no Python, no server, no API key required
Step-by-step in Claude Cowork
  1. Open claude.ai and click Cowork in the left sidebar.
  2. Click New Automation and set the schedule to Daily at your preferred time (e.g. 08:00).
  3. In the "What should Claude do?" field, paste the prompt below.
  4. Replace [YOUR_URL] with the page you want to scrape and [YOUR_CSS_SELECTOR] with the element to extract (e.g. span.price, h2.headline).
  5. In the output settings, choose Append to Google Sheet (or download as CSV) and name the sheet scraped_data.
  6. Click Save & Activate. Claude will run the scrape every day at the scheduled time.
You are a daily web scraping automation. Each time you run, do the following:

1. Fetch the webpage at: [YOUR_URL]
2. Extract all text content from elements matching CSS selector: [YOUR_CSS_SELECTOR]
3. Clean each value — strip leading/trailing whitespace and remove stray quotation marks.
4. For each extracted value, output one row in this exact CSV format:
   timestamp,url,selector,value
   Use ISO-8601 for the timestamp (YYYY-MM-DD HH:MM:SS).
5. After the CSV rows, print a short summary:
   - How many rows were extracted
   - The first 3 values (truncated to 80 chars each)
   - Any errors encountered

Output ONLY the CSV rows followed by the summary. Do not add any other text.

Reference: local Python version
The file main.py in this folder is the original local Python implementation. It requires pip install -r requirements.txt, a .env file, and a cron job on your machine. The Cowork guide above achieves the same result without any local setup.
