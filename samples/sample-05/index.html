Hands-On 14 · Sample 05

Two-Source Sync Job

Keep two data sources in sync automatically — find records present in Source A but missing from Source B, and append them with a timestamp and change log.

Subscription required

Claude.ai Pro or Team plan with Cowork access. No API key or local Python setup needed.

What this does

This automation compares two data sources (two Google Sheets, a CRM export and a local sheet, or any two CSV-based lists), identifies rows present in the primary source (A) but missing from the secondary source (B), and appends only those new rows to B — with a sync timestamp and delta log.

  • Reads both sources and extracts ID values from the configured key column
  • Computes the delta: IDs in A not yet in B
  • Appends missing rows to Source B with a __synced_at timestamp column
  • Writes a delta log entry recording exactly what changed and when
  • Idempotent: running twice in a row makes no changes on the second run
Subscription required

This guide uses Claude Cowork — a scheduled automation feature available on the Claude.ai Pro and Team plans.

  • Go to claude.ai and sign in
  • You need a Pro or Team subscription
  • Connect your Google Sheets (or Google Drive) in Cowork settings so Claude can read both sources and write to the secondary sheet
Step-by-step in Claude Cowork
  1. Open claude.ai and click Cowork in the left sidebar.
  2. In Cowork settings, connect your Google Sheets account. Make sure both your Source A sheet and Source B sheet are accessible.
  3. Create a third sheet called delta_log — Claude will append one row per sync run here as your change history.
  4. Click New Automation and set the schedule to Hourly or Daily depending on how often your primary source updates.
  5. Paste the prompt below, replacing the sheet IDs, tab names, and key column name with your own values.
  6. Click Save & Activate. After each run, only genuinely new records will be added to Source B.
You are a two-source sync automation. Each time you run, do the following:

SOURCES:
- Source A (primary/authoritative): Google Sheet ID [SOURCE_A_SHEET_ID], tab "[SOURCE_A_TAB]"
- Source B (secondary/destination): Google Sheet ID [SOURCE_B_SHEET_ID], tab "[SOURCE_B_TAB]"
- Key column (unique ID field): [KEY_COLUMN_NAME]
- Delta log: Google Sheet ID [DELTA_LOG_SHEET_ID], tab "log"

STEPS:

1. Read all rows from Source A. Use the first row as the header.
   Build a list of all values in the key column → call this SET_A.

2. Read all rows from Source B. Use the first row as the header (or create it if the sheet is empty).
   Build a list of all values in the key column → call this SET_B.

3. Compute the delta: rows in Source A where the key value is NOT in SET_B.
   Call this DELTA_ROWS.

4. If DELTA_ROWS is empty:
   - Output: "Sync complete — no new records. Source A: [count_a] rows. Source B: [count_b] rows."
   - Stop here.

5. If DELTA_ROWS is not empty:
   a. Append each row in DELTA_ROWS to Source B, adding a new column "__synced_at" with the current UTC timestamp (YYYY-MM-DD HH:MM:SS) as the last column.
   b. Append one row to the delta log sheet with these columns:
      run_at, source_a_rows, source_b_rows_before, synced_count, synced_ids
      (synced_ids: comma-separated list of the key values that were synced)

6. Output a summary:
   "Sync complete — [count] new records added to Source B."
   "Synced IDs: [comma-separated key values]"
   "Source A: [count_a] rows | Source B was: [count_b_before] → now: [count_b_after] rows"
Sample output
=======================================================
  Sync Job -- Run Complete
=======================================================
  Source A     : source_a.csv (4 rows)
  Source B was : source_b.csv (2 rows)
  Delta        : 2 new records

  Synced IDs:
    + ID-003 (Alice Johnson)
    + ID-004 (Bob Williams)

  Source B now : 4 rows
  Delta log    : 1 entry appended

Second run (idempotent check):
  Source A     : 4 rows
  Source B     : 4 rows
  Delta        : 0 new records
  Sync complete — no new records.
=======================================================
Reference: local Python version
The file main.py in this folder is the original local Python implementation. It reads from source_a.csv and source_b.csv on the local filesystem and writes a delta_log.jsonl file. The Cowork guide above achieves the same result using two Google Sheets as the data sources, with no local files or Python environment needed.
